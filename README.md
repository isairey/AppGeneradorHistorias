# 🌌 Acerca de StoryForge AI

**StoryForge AI** es una plataforma impulsada por Inteligencia Artificial diseñada para generar historias originales a partir de palabras clave, géneros, personajes y escenarios definidos por el usuario.

La plataforma permite crear cuentos infantiles, historias de fantasía, relatos de ciencia ficción, novelas románticas, aventuras épicas y mucho más mediante modelos avanzados de procesamiento de lenguaje natural.

Ideal para:

- ✍️ Escritores
- 📚 Estudiantes
- 🎨 Creadores de contenido
- 🎮 Desarrolladores de videojuegos
- 📖 Lectores y aficionados a la literatura

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/AppGeneradorHistorias
```

---

## 2️⃣ Instalar dependencias

```bash
npm install
```

---

## 3️⃣ Configurar variables de entorno

```env
VITE_OPENAI_API_KEY=your_api_key
VITE_API_URL=http://localhost:3000
```

---

## 4️⃣ Ejecutar aplicación

```bash
npm run dev
```

---

## 5️⃣ Abrir navegador

```bash
http://localhost:5173
```
